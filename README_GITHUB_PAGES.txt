# Shams Wasfaty – GitHub Pages Package

This package contains a single-page site ready for GitHub Pages.

## Files
- index.html  ← deploy this at the repository root

## Deploy (GitHub Pages)
1) Create a new public repo, e.g. `shams-wasfaty`
2) Upload `index.html` to the repository root and commit
3) Go to: Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
   - Save
4) Your site will be available at: https://USERNAME.github.io/shams-wasfaty/
